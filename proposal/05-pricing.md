# Pricing

**Submitted by:** Accenture
**RFP Reference:** MC-2026-0417

---

## Pricing Structure

Accenture proposes a **fixed-fee engagement** for the base scope (R1–R4), with optional extensions for desired items (D1–D3) priced separately. Fixed-fee pricing gives Meridian cost certainty and removes the ambiguity that T&M structures introduce when working from an incomplete defect list.

---

## Base Engagement (R1–R4)

| Phase     | Scope                                                     | Fixed Fee   |
|-----------|-----------------------------------------------------------|-------------|
| Phase 1   | Architecture documentation (R4) + Reports audit (R1)      | $18,000     |
| Phase 2   | Reports remediation (R1) + browser tests for Reports (R3) | $32,000     |
| Phase 3   | Restocking view (R2) + browser tests for Restocking (R3)  | $48,000     |
| Phase 4   | Hardening, handoff                                        | Included    |
| **Total** |                                                           | **$98,000** |

Payment schedule: 25% on contract execution, 25% at Phase 1 milestone, 25% at Phase 2 milestone, 25% at Phase 3/final milestone.

---

## Optional Extensions (D1–D3)

| Item                      | Scope                                               | Estimated Fee   |
|---------------------------|-----------------------------------------------------|-----------------|
| D1 — UI Modernization     | Design token refresh, component styling update      | $18,000–$24,000 |
| D2 — Internationalization | i18n extension to remaining modules, Tokyo priority | $12,000–$16,000 |
| D3 — Dark Mode            | Operator-selectable theme via CSS custom properties | $8,000–$12,000  |

Extensions are estimated ranges pending scope confirmation. Each can be contracted independently following base engagement delivery.

---

## Pricing Assumptions

| #  | Assumption                                                                                                             |
|----|------------------------------------------------------------------------------------------------------------------------|
| P1 | Engagement is staff-augmentation style: one senior engineer, part-time engagement manager                              |
| P2 | Meridian provides access to the codebase, test environment, and a point of contact for questions within 1 business day |
| P3 | No infrastructure changes are in scope (no database migration, no hosting changes)                                     |
| P4 | Feedback turnaround from Meridian IT on milestone deliverables is within 3 business days                               |
| P5 | D1 estimate assumes a moderate visual refresh; a full redesign would require re-scoping                                |

---

## Not-to-Exceed

The base engagement fixed fee of **$98,000** is the not-to-exceed amount for R1–R4. Accenture absorbs any overrun within base scope. Out-of-scope work (including D1–D3 or defects surfaced in modules outside Reports) will be flagged and quoted before work begins.
